## About Us

**TechCoderHub** is a specialized Qt/QML engineering studio based in Gurugram, India — serving clients worldwide. We build production-grade desktop applications, automotive dashboards, industrial HMI systems, home automation interfaces, and embedded device UIs using the Qt framework.

We work with startups, product teams, and engineering groups who need practical Qt engineering, direct technical collaboration, and maintainable source delivery.

---

## What We Do

| Service | Description |
|:--------|:------------|
| **Qt/QML & C++ Development** | Custom Qt Quick applications for desktop, embedded, and cross-platform targets |
| **HMI & Dashboard Development** | Automotive instrument clusters, infotainment systems, and industrial control panels |
| **Embedded Qt & Boot2Qt** | Qt on embedded Linux, Yocto integration, EGLFS, and device bring-up |
| **Qt for MCU** | Qt Quick Ultralite interfaces on ARM Cortex-M microcontrollers |
| **QWidget to QML Migration** | Incremental modernization of legacy Qt applications |
| **Desktop Application Development** | Cross-platform desktop tools with modern QML interfaces |
| **Custom Software Development** | End-to-end product development from prototype to production |
